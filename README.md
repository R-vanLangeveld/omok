# Overview

I've recreated the table top game Omok from the video game Vintage Story. Omok is like if connect four didn't have gravity, you needed to connect five pieces to win, and if you played on a board with a 15x15 grid. The current version of my recreation is best experienced with two players sitting next to eachother infront of the same computer.

I created this game to learn C++.

[Software Demo Video](https://youtu.be/m7XQn4EvpiU)

# Development Environment

The code editor I used to program this game was VSCode.

I used C++, an object-oriented programming language, to program this game.
